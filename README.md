# 🟢 Pokédex

Uma Pokédex desenvolvida utilizando **HTML, CSS e JavaScript puro**, consumindo dados da **PokéAPI**.

Este é o meu primeiro projeto desenvolvido do zero. Durante o desenvolvimento procurei colocar em prática conceitos de manipulação do DOM, consumo de APIs, programação assíncrona e organização do código em módulos ES6.

Embora ainda existam pontos que podem ser melhorados e refatorados, decidi considerar este projeto como finalizado por representar minha evolução como desenvolvedor Front-end.

---

## 🚀 Demonstração

🔗 **Acesse o projeto:**
**https://renanstein.github.io/Pokemon-Pokedex/**

---

## ✨ Funcionalidades

* Listagem de Pokémons consumindo dados da PokéAPI
* Busca por nome ou número do Pokémon
* Filtro por tipo
* Carregamento incremental através do botão **Load More**
* Layout totalmente responsivo
* Menu responsivo para dispositivos móveis
* Animação dinâmica no banner principal
* Tratamento de erros durante as requisições da API

---

## 🛠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API
* Async/Await
* Promise.all()
* ES Modules
* PokéAPI

## 📚 O que aprendi durante este projeto

Este projeto foi um grande desafio e me permitiu colocar em prática diversos conceitos importantes, como:

* Manipulação do DOM
* Consumo de APIs REST
* Organização do código em módulos
* Programação assíncrona com Async/Await
* Utilização de Promise.all()
* Tratamento de erros com try/catch
* Eventos e interação com o usuário
* Responsividade utilizando CSS
* Estruturação de uma aplicação JavaScript sem frameworks

Além da parte técnica, este projeto também fortaleceu minha capacidade de pesquisar, depurar problemas e encontrar soluções para desafios que surgiram durante o desenvolvimento.
